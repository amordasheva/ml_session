# Signal Types Classification

Решение задачи кластеризации сигналов сцинтилляционного детектора для Kaggle-соревнования:
https://www.kaggle.com/competitions/signal-types-classification

## Файлы

* `Signal_clustering_solution.ipynb` — ноутбук с решением;
* `kaggle_results.png` — скриншот результата на Kaggle.

## О решении

В работе выполнены анализ сигналов, предобработка данных и выделение признаков, связанных с формой импульса: амплитуда, площадь, baseline, PSD и параметры хвоста.

Для финального разделения использованы два основных физических кластера. Явные аномалии были проанализированы отдельно, но их выделение в третий кластер ухудшало результат.
